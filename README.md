# Лабораторна робота 1: Налаштування шаблону проекту

## Опис
Базовий Flask додаток з ендпоінтом healthcheck для лабораторної роботи з дисципліни "Технології серверного програмного забезпечення".

## Виконавець
Бондар Ярослав ІМ-33

Варіант: **3 - Облік доходів**

## Технології
- Python 3.11
- Flask 2.3.3
- Gunicorn
- Docker
- Docker Compose

## API Ендпоїнти

### GET /
Повертає вітальне повідомлення.

**Приклад відповіді:**

```json
{
  "message": "Welcome to the Flask API!"
}
```
###
GET /healthcheck

Повертає статус сервісу та поточну дату.

**Приклад відповіді:**

```json
{
  "date": "2023-10-10T14:30:00Z",
  "status": "healthy"
}
```
